<h3>Follow these steps to execute the project.</h3>

STEP 1: Clone the repository and save it to your local memory.

STEP 2: Open the directory name "Backend_NodeJS" in VsCode.

STEP 3: Now Open Terminal on VsCode and first run npm install and then run nodemon index.js command.

STEP 4: Open the directory name "Frontend_Angular" in VsCode(new).

STEP 5: Now Open Terminal on VsCode and first run npm install and then run ng serve command.

STEP 6:Now GO to your local host and Enter details as you see on webpage and run application.
